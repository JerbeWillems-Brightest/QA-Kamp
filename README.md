# QA-Kamp
Een QA-kamp, waar jongeren kinderen tussen 8 en 16 jaar spelletjes spelen en zich kunnen verdiepen in de wereld van IT en softwaretesting.

# Gebruik
Backend:
- Ga naar ./QA-Kamp/Backend
- Run: npm install
- Run: npm run dev

Frontend:
- Ga naar ./QA-Kamp/Frontend/QA-Kamp
- Run: npm install
- Run: npm run dev
